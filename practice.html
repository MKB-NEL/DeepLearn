HTML Certification Exam

Complete all questions to earn your HTML certification. Each question builds on the previous ones.

1 Document Structure

Create a proper HTML5 document structure with:

  • Correct DOCTYPE declaration
  • HTML element with lang attribute
  • Head section with UTF-8 charset meta tag and title
  • Body section

2 Semantic Page Layout

Expand your document to include semantic elements:

  • A header with a main heading
  • A navigation section with 3 links
  • A main content area with an article
  • A footer with copyright information

3 Content Markup

Add content to your article section with:

  • A heading for the article
  • Two paragraphs of text
  • An unordered list with 3 items
  • An image with proper alt text
  • A link to an external website that opens in new tab

4 Table Creation

Add a table to your document with:

  • A caption
  • Table headers in the first row
  • 3 rows of data
  • Use appropriate semantic elements (thead, tbody, etc.)

5 Form Creation

Add a form to your document with:

  • Text input for name (required)
  • Email input with placeholder
  • Dropdown for country selection
  • Checkbox for newsletter subscription
  • Textarea for comments
  • Submit and reset buttons
  • Proper labels for all fields

6 Attributes & Styling

Enhance your document with:

  • Add id and class attributes to at least 3 elements
  • Use style attribute to change color of one element
  • Add a meta viewport tag for mobile devices
  • Add a favicon link
