Basic HTML Tags

HTML documents are made up of elements defined by tags. Here are the fundamental building blocks:

Headings

Headings define the hierarchy of your content from <h1> (most important) to <h6> (least important).

<h1>Main Title</h1>
<h2>Subheading</h2>
<h3>Section Heading</h3>

Paragraphs

The <p> tag defines a paragraph of text.

<p>This is a paragraph of text.</p>
<p>This is another paragraph.</p>

Links

The <a> tag creates hyperlinks with the href attribute.

<a href="https://example.com">Visit Example</a>
<a href="about.html" target="_blank">About Us (opens in new tab)</a>

Images

The <img> tag embeds images with src and alt attributes.

<img src="logo.png" alt="Company Logo">
<img src="photo.jpg" alt="Beautiful landscape" width="400" height="300">

Lists

HTML supports both ordered (<ol>) and unordered (<ul>) lists.

<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

<ol>
  <li>Step one</li>
  <li>Step two</li>
</ol>

Page Structure

Proper HTML document structure is essential for browser compatibility and SEO.

Basic Structure

Every HTML document should have these essential elements:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <!-- Content goes here -->
</body>
</html>

Head Elements

The <head> contains metadata and resources:

<meta name="description" content="Page description">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="styles.css">
<style>
  body { font-family: Arial; }
</style>
<script src="script.js" defer></script>

Forms & Input

HTML forms collect user input and send it to a server for processing.

Basic Form Structure

<form action="/submit" method="POST">
  <!-- Form elements go here -->
  <button type="submit">Submit</button>
</form>

Input Elements

<input type="text" name="username" placeholder="Enter username" required>
<input type="email" name="email" placeholder="Your email">
<input type="password" name="pwd" placeholder="Password">
<input type="number" name="age" min="18" max="99">
<input type="date" name="birthday">
<input type="checkbox" name="subscribe" id="subs" checked>
<label for="subs">Subscribe to newsletter</label>

Other Form Elements

<textarea name="message" rows="4" placeholder="Your message"></textarea>

<select name="country">
  <option value="">Select country</option>
  <option value="us">United States</option>
  <option value="ca">Canada</option>
</select>

<button type="submit">Submit Form</button>
<button type="reset">Reset</button>

HTML Tables

Tables organize data into rows and columns.

Basic Table Structure

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Row 1, Cell 1</td>
    <td>Row 1, Cell 2</td>
  </tr>
</table>

Advanced Table Features

<table>
  <caption>Monthly Sales</caption>
  <thead>
    <tr>
      <th colspan="2">2023 Sales</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>January</td>
      <td>$10,000</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Total</td>
      <td>$10,000</td>
    </tr>
  </tfoot>
</table>

Semantic HTML

Semantic HTML improves accessibility and SEO by clearly describing content meaning.

Common Semantic Tags

<header>
  <h1>Website Title</h1>
  <nav>
    <!-- Navigation links -->
  </nav>
</header>

<main>
  <article>
    <h2>Article Title</h2>
    <section>
      <h3>Section Heading</h3>
      <p>Content...</p>
    </section>
  </article>
</main>

<aside>
  <!-- Sidebar content -->
</aside>

<footer>
  <p>Copyright information</p>
</footer>

When to Use Semantic Elements

  • <header> - Introductory content or navigation
  • <nav> - Major navigation blocks
  • <main> - The dominant content of the document
  • <article> - Self-contained composition
  • <section> - Thematic grouping of content
  • <aside> - Content tangentially related to main content
  • <footer> - Footer for its nearest section

HTML Attributes

Attributes provide additional information about HTML elements.

Common Attributes

<!-- id - Unique identifier -->
<div id="main-content"></div>

<!-- class - For styling and JavaScript -->
<p class="important note">Important note</p>

<!-- style - Inline CSS -->
<h1 style="color: blue;">Blue Heading</h1>

<!-- data-* - Custom data attributes -->
<div data-user-id="123"></div>

Element-Specific Attributes

<img src="image.jpg" alt="Description" width="300" height="200">

<a href="page.html" target="_blank" rel="noopener">Link</a>

<input type="text" name="username" placeholder="Enter name" required disabled>

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Boolean Attributes

Attributes like disabled, readonly, required, and checked don't need a value:

<input type="checkbox" checked>
<button disabled>Can't click</button>
<input type="text" readonly value="Can't edit">
